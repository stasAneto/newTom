Центр стоматологической диагностики NewTom GO 2D/3D

Цифровой дентальный рентгенаппарат NewTom GO 2D/3D это самая новая концепция для стоматологической практики, сочетающая в себе простоту использования, надежность и диагностическое качество обследований. Для минимизации дозы облучения применятся технология SafeBeam с автоматической адаптацией параметров экспозиции. Для оптимизации качества получаемых снимков (четкости, яркости и контраста) применяется технология ApT. Специальные режимы 3D ECOScan и 2D ECOPan позволяют при необходимости еще больше снизить дозу облучения (время сканирование 6,4 сек).

Для повышения диагностического качества ОПГ применяется технология динамического автофокуса MultiPAN при которой за одно сканирование формируется набор из пяти различных панорамных изображений на разных слоях с шагом 1 мм. 3D-сканирование осуществляется с очень высоким разрешением (воксель 0,08 мм) с размерами области сканирования от 10х10 до 6х6 см и режимом предварительного прицеливания для избежания ошибок позиционирования головы и повторного сканирования. Кроме того для точности позиционирования головы прицеливание осуществляется 3-мя лазерными лучами.

Для уменьшения влияния артефактов наличия металла во рту применяется фильтр aMAR (опция).

Просмотр полученных результатов на диске, возможен без предварительной установки «NNT Viewer».

Приемный сенсор (CSI) с динамическим диапазоном 16 бит (65.536 оттенков серого). Генератор постоянного тока DC с трубкой 60~90 кВ/ 4~15 мА, фокусное пятно 0,5 мм.
